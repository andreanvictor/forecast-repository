Project.config_dict

When creating a new Project, you must specify a config_dict. It is a JSON object that specifies project-specific information using these keys:

  1. target_to_week_increment: an object that maps week-related target names to ints, such as '1 wk ahead' → 1 . Also, this objects's keys are used by the Mean Absolute Error method to decide which targets to use.
  2. location_to_delphi_region: an object that maps all my locations to Delphi region names, as documented here.
